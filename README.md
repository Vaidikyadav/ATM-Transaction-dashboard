🏧 ATM Transaction Dashboard (Power BI)

This project presents an interactive ATM Transaction Dashboard built using Power BI, designed to analyze ATM operations, costs, revenues, and performance across multiple regions.

📊 Dashboard Overview

The dashboard provides insights into:

💰 Cost Analysis – Breakdown of ATM maintenance costs (CRA, AMC, Site Maintenance, UPS, VSAT, Spare Replacement)
📈 Revenue Analysis – Revenue generated from ATMs and MHA across different states
🔄 Transaction Analysis – Comparison of Financial vs Non-Financial transactions by month
📅 Trend Analysis – Monthly transaction trends over time
📍 Regional Performance – State-wise revenue and performance metrics
🔑 Key Metrics
Total Cost: 296M
Average Transactions: ~179M
Gross Profit: ~59%
Average Uptime: ~92%
Average EBILL: ~2.2K
